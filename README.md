<h1 align="center">Nexoura</h1>

<p align="center">
Electronics E-commerce Website
</p>
A electronics e-commerce website built with HTML, CSS and JavaScript.

The project demonstrates a modern online shopping experience with product browsing, product details and shopping cart management using LocalStorage.

---

## Live Demo

🔗 https://mhtbdni.github.io/Nexoura/

---

## Features

- Product listing
- Product details page
- Image gallery
- Color selection
- Quantity selection
- Shopping cart
- Client-side form validation
- LocalStorage persistence
- Price and discount calculation
- Responsive design
  

---

## Technologies

- HTML5
- CSS3
- JavaScript
- JSON
- Fetch API
- LocalStorage
